## Collaborative Engagement and Processing Unit
A code base for a project for students' engagement measuring during classes.
It records audio via microphones connected to a RaspberryPI microcontroler and transfer the data to a local HTTP / MQTT server for processing.
Measures the volume of the conversation, distribution of speaker time and on-topicness of the conversation.
Presents the data in and easy-to-use dashboard.
